# Resource GDPR AYX

本项目用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为技术性资源存档使用。

## 项目简介

这是一个静态资源归档仓库，主要存放若干独立 HTML 页面。页面内容可能涉及通用信息展示、工具页面或演示用途，不包含特定商业或营销内容。仓库本身不提供任何形式的服务或功能，仅作为文件存储与版本管理。

## 目录说明

```
.
├── index.html          # 首页入口文件（如存在）
├── pages/              # 存放独立 HTML 页面
│   └── example.html    # 示例页面（实际以仓库内容为准）
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

各目录作用：

- `pages/`：存放所有独立 HTML 页面，每个页面可单独访问。
- `assets/`：存放页面所需的静态资源，按类型分目录管理。

*实际目录结构以仓库最新提交为准。*

## 页面归档说明

本仓库中的 HTML 页面均为独立文件，彼此之间无强依赖关系。每个页面可单独打开使用，不需要后端服务支持。页面可能引用公共资源（如 CSS、JavaScript 文件），这些资源存放在 `assets/` 目录中。

归档内容可能不定期更新或清理，不保证所有页面长期可用。建议使用者根据自身需求自行备份。

## 维护说明

- 页面内容可能因技术调整或归档需要被修改或删除。

## 使用方式

1. 克隆仓库到本地：
   ```
   git clone https://github.com/your-username/resource-gdpr-ayx.git
   ```
2. 直接打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
